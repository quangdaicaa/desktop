## %Cpu(s):
  
|     |              |                    |         |
| --- | ------------ | ------------------ | ------- |
| us  | user         | Người dùng sử dụng | Lục     |
| sy  | system       | Root sử dụng       | Đỏ      |
| ni  | user nice    | Ưu tiên thấp       | Lam     |
| id  | idle         | Rảnh               |
| wa  | io wait      | Đợi                | Xám     |
| hi  | hardware irq | Sửa phần cứng      | Vàng    |
| si  | software irq | Sửa phần mềm       | Đỏ tươi |
| st  | steal time   | Máy ảo             |

|      |         |
| ---- | ------- |
